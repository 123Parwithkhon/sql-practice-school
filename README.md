# 📊 SQL School Database Practice

Коллекция SQL-скриптов для решения аналитических задач в рамках базы данных образовательного учреждения (SCHOOL).

## 🎯 Что реализовано в проекте:
В ходе выполнения задач были проработаны следующие SQL-концепции:
* **Сложные объединения:** Использование `LEFT JOIN`, `RIGHT JOIN` и `INNER JOIN` для сборки данных из 4+ таблиц.
* **Агрегация и фильтрация:** Использование `GROUP BY`, `COUNT` и `HAVING` для получения статистики.
* **Подзапросы (Subqueries):** Сравнение индивидуальных показателей со средними значениями по базе (например, поиск учеников с оценкой выше средней).
* **Управление данными (DML):** Операции вставки (`INSERT`) и выборки для миграции данных.

## 📋 Список решенных бизнес-задач:
1. Контроль заполняемости журналов (поиск пустых оценок).
2. Анализ нагрузки преподавательского состава.
3. Мониторинг использования учебных кабинетов.
4. Выявление отличников на основе статистического среднего.

## 🛠 Стек технологий
- **SQL** (PostgreSQL / MySQL / MS SQL Server)
- Работа с реляционными связями и целостностью данных.
